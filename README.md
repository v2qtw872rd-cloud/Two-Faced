# Two-Faced
Is studying hard for you? Do you like a challenge? If so, then this will be quite the experience for you! Introducing "Two-Faced," the star of the show that's a dual-mode application where users can study and challenge themselves all in one! Two-Faced is a tool with intermediate concepts such as classes, various modules, and "Pygame."
Two-Faced has multiple files which are "Assets," an image folder, "challene_images.py," "image_reveal.py," "main.py," "modes.py," and "two_faced_timer.py." The main called "main.py" is like the control center, while all the other files are functions and data that's imported into the main file. In addition, this application uses concepts such as "Object-Oriented Programming (OOP)", functions, modules such as "os" and "pygame," condtitional statements, and much more for a coherent and finished project!
